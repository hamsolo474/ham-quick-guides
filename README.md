# ham-quick-guides
Quick guide creation for GIMP

Install by putting ham-quick-guides.py into your Gimp plugins folder

https://thegimptutorials.com/how-to-install-gimp-plugins/

Run by going to Image > Edit > Guides > Quick Guides
